### PRM

实验要求: 绿色方块代表起始位置，红色方块代表目标位置，要求在已知地图全局信息的情况下，规划一条尽可
能短的轨迹，控制机器人从绿色走到红色

解决方法：先用 PRM 算法规划最短路径，并将其标记在图中，并用上次的自动巡线代码控制小车沿着线走。
